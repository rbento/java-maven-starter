# java-maven-starter

A starter template for Java SE projects managed with Maven.

### Dependencies
---

- [Java][1]
- [Maven][2]

[1]: https://openjdk.org/projects/jdk/20/
[2]: https://maven.apache.org/download.cgi

### Build
---

```bash
mvn clean package
```

### Run
---

```bash
mvn exec:java -Dexec.mainClass="rbento.Program"
```
